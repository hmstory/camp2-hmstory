# 보이스 그룹 카운슬링 앱 — PRD

## 한 줄 요약

AI 대화로 고민을 파악하고, 비슷한 사람끼리 자동 매칭하여, 자격 있는 상담사가 이끄는 음성 그룹 상담을 연결하는 서비스.

## 문제

4명 중 1명이 정신건강 문제를 경험하지만(WHO, 2022), 한국 성인의 서비스 이용률은 7.2%(국립정신건강센터, 2021)에 불과하다. OECD 최하위 수준의 정신건강 접근성. 비용, 접근성, 낙인 — 세 가지 벽이 상담을 가로막고 있다.

- **현재 상태**: 기존 서비스는 전부 1:1 중심이고, 그룹 매칭은 사람이 수동으로 한다. AI는 상담사 추천에만 사용됨.
- **원하는 상태**: AI와 대화하면 EBT 기반 척도로 고민 주제와 심각도를 평가하고, 비슷한 사람끼리 소그룹(4~5명)을 자동 구성. 자격 있는 상담사가 음성으로 그룹 세션을 진행.

## 왜 보이스인가

| 이유 | 설명 |
|------|------|
| 얼굴 부담 없이 | 화면 켤 필요 없음. 목소리만으로 충분히 연결됨 |
| 밤 11시에도 | 퇴근 후, 아이 재운 후. 내가 편한 시간에 참여 |
| 어디서든 편하게 | 침대에서, 산책하면서. 장소 제약 없음 |

## 경쟁 분석

| | [Talkspace](https://medcitynews.com/2025/07/talkspace-tia-women-mental-health/) | [BetterHelp](https://www.betterhelp.com/about/) | [트로스트](https://www.trost.co.kr/) | 우리 |
|--|-----------|------------|---------|------|
| 상담 형태 | 1:1 텍스트/화상 | 1:1 텍스트/화상 | 1:1 텍스트 | **그룹 음성** |
| 매칭 방식 | 의사 수동 의뢰 | 설문 -> AI 추천 | 사용자 직접 검색 | **AI 대화 -> 자동 그룹** |
| AI 역할 | 없음 | 상담사 추천만 | 없음 | **평가 + 분류 + 매칭** |
| 그룹 상담 | 없음 | 있음 (수동 배정) | 없음 | **AI 자동 매칭** |

**핵심 차별점**: AI가 매칭하고, 그룹으로 연결하는 서비스는 아직 없다.

## 핵심 인사이트

> "AI 챗봇은 결국 AI다. 우리는 AI를 도구로 써서 사람과 사람을 연결한다."

AI를 치료 도구가 아닌 **연결 도구**로 사용하는 것이 이 프로젝트의 핵심 관점.

## 3단계 로드맵

| Phase | 내용 | 개발 방식 | 상태 |
|-------|------|----------|------|
| **1단계** | EBT 기반 대화형 심리 평가 챗봇 | 수업 전용 플랫폼 (Prompt Engineering + RAG) | 진행 중 |
| **2단계** | AI 그루핑 엔진 (평가 결과 -> 자동 매칭) | 자체 개발 | 예정 |
| **3단계** | 상담사가 이끄는 소그룹 음성 집단 상담 | 자체 개발 | 엔드 픽쳐 |

## Phase 1 상세: 그룹 매칭을 위한 EBT 기반 심리 평가 챗봇

### 목적
단순 상담 챗봇이 아닌, **2단계 그루핑의 입력값을 생성하는 평가 챗봇**.

### 설계

| 항목 | 내용 |
|------|------|
| EBT 선택 | TBD — 수업에서 배운 후 교수님과 상의하여 확정 |
| 대화 구조 | 단계별: 스크리닝 대화 -> 관심 시 심층 평가 |
| 출력 | 고민 카테고리 + 심각도 + 호소문제 요약 (구조화된 결과) |
| 분류 체계 | TBD — EBT 확정 후 결정 (EBT에 따라 분류 기준 달라짐) |

### 챗봇 파이프라인

```
사용자 대화 -> EBT 기반 평가 -> 구조화된 결과 출력 -> 2단계 그루핑 입력
```

### 안전장치
- 사전 동의: "본 서비스는 위기 상담을 대체하지 않습니다"
- 위기 연계처(1393) 상시 표시
